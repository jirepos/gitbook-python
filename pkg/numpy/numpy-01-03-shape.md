# Numpy

## numpy.ndarray.shape

shape 속성은 일반적으로 배열의 현재 모양을 가져오는 데 사용되지만 배열 차원의 튜플을 할당하여 배열의 모양을 제자리에서 변경하는 데 사용할 수도 있습니다.

```
import numpy as np 
```

```
x = np.array([1, 2, 3, 4])
x.shape
```
```
(4,)
```
아래는 2 X 3 X 4의 배열을 생성한다. shape (2,3,4)는  2 X 3 X 4의 배열임을 보여주고 있다.

```
y = np.zeros((2, 3, 4))
y.shape
```
```
(2, 3, 4)
```
